Hexo 安装部署

  1. 本地安装 Git，Node.js，hexo
  2. 配置 Hexo

本地安装 Git，Node.js，hexo

  • Git 安装：官网教程

  • Node.js 下载安装：官网

  • Hexo 安装：官网

配置 Hexo

初始化 hexo

  • 创建目录

  • 初始化 hexo 或者直接进入现有目录

  • 运行 hexo

配置 个性化 文件

  • 先修改 hexo 文件目录下的 _config.yml，配置全局属性

  • 下载 next 主题，并放到 theme 文件夹下

  • 修改 next 主题目录下 _config.yml，配置主题属性

链接到 github，coding

安装 hexo-deployer-git

安装完成后，将如下配置添加到 _config.yml 文件中

其中<repo>为刚创建的Github Pages仓库的地址，格式为：https://github.com/你的Github用户名/仓库名.git

配置完成后，运行命令hexo g或hexo generate生成静态文件(如果觉得文件有缓存，还可以先运行hexo clean命令，再运行hexo g)

运行 hexo d 后会提示输入github账号及密码，输入后回车即可。

默认github page 地址：databian.github.io
