Debian 安装 wordpress

获取 root 权限

安装 yum 和 wget

屏幕监视程序

  • 如果提示screen: command not found 命令不存在可以执行：yum install screen 或 apt-get install screen安装，如果VPS连接中断，可以使用命令screen -R lnmp返回会话窗口。

安装 lnmp 一键脚本

  • 选择配置见lnmp教程
  • 安装完成后运行killall screen关闭监视窗口

创建虚拟主机，配置数据库

  • 选项配置见lnmp教程或搬瓦工教程

下载解压 wordpress 并移动文件至网站根目录

  • 以上命令可以用 && 连接为一整个脚本

  • wget https://wordpress.org/latest.tar.gz && tar -xzvf latest.tar.gz && cd wordpress && mv * /home/wwwroot/www.databian.com/

  • 提示chown: changing ownership of /home/wwwroot/www.baidu.com/.user.ini': Operation not permitted 执行以下命令. sed -i 's/,scandir//g' /usr/local/php/etc/php.ini && lnmp php-fpm restart

将域名解析到服务器，输入域名即可开始配置。
