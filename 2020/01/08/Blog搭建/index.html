Blog 搭建

发表于 2020-01-08 更新于 2020-03-26

博客的最佳运营维护方式：本地markdown文本编辑 - 云盘备份md内容 - 推流到 gitpage & wordpress

  • 本地使用 Typora 编辑 Markdown 博文

  • .md 博文存储于本地，定期备份到 OnedDrive

  • GitHub Page && Codding Page 搭建 Hexo 博客

  • VPS 搭建 Wordpress 服务，并绑定域名

  • 博文推流到 Hexo && wordpress

博客搭建有多种方式，最简单的就是用现成的博客社区，但是这些社区无法做个性化定制，而且国内言论限制严格，诸多不变。在私人服务器上搭建博客虽然比较折腾，但长远来看更有利于展现完整的思想。以下有四个博客展示，wordpress和hexo是比较推荐的。特别是hexo，使用markdown语言编辑，直接生成为网站，效率高速度快。
